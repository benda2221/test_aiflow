# Header Files Directory

This directory contains C header files defining the public API for the SDK.

## Organization

- `hello.h`: Header for hello function
- `cholesky.h`: Header for Cholesky decomposition function

## Key Files

- `cholesky.h`: Declares `cholesky_decompose_32x32()` function
  - Implementation: `../src/cholesky.c`
  - Documentation: `../docs/cholesky.md` and `../src/cholesky.md`
